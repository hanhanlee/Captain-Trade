## 第十四章：每日標準操作流程

### 開盤前（早上 9:00 前）

```
✅ 步驟 1：啟動工具
   → 開啟黑色視窗，輸入 streamlit run app.py

✅ 步驟 2：查看市場環境
   → 確認今天大盤是多頭/偏多/偏空/空頭
   → 這決定今天是積極操作還是保守觀望

✅ 步驟 3：查看持股監控
   → 確認昨天收盤後有無新的警示
   → 如有警示，決定今天是否需要採取行動
```

### 盤中（上午 9:00 — 下午 1:30）

```
📲 LINE 在 13:30 會自動推播盤中持股警示
   → 收到警示時，進入工具查看詳情
   → 依照警示等級和自己的判斷決定是否處理
```

### 收盤後（下午 2:30 — 4:00）

```
📲 LINE 在 14:35 推播盤後警示、14:45 推播選股結果

✅ 步驟 1：研究選股結果
   → 看 K 線圖確認入選股票的型態
   → 找出合理的進場點和停損位置

✅ 步驟 2：計算部位大小
   → 前往「風險控制」頁面
   → 輸入進場價、停損價，取得建議張數

✅ 步驟 3：記錄今日交易
   → 如果今天有買賣，前往「交易日誌」記錄
   → 包含理由和當時的情緒
```

### 每週五收盤後（特別流程）

```
📲 LINE 在 15:10 推播週績效摘要

✅ 週回顧流程：
   1. 查看週績效：這週勝率如何？盈虧比如何？
   2. 回顧每筆交易：哪筆做得好？哪筆做得不好？
   3. 情緒分析：哪種情緒下的交易表現最差？
   4. 調整計畫：下週的操作策略有什麼需要調整的？
```
