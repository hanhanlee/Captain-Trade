## 第八章：選股雷達 — 找值得買的股票

### 功能說明

每天掃描台股市場（可選 20 / 100 / 全市場檔），依技術分析條件篩選出符合標準的股票，依強度分數排名輸出。

### 掃描範圍選擇建議

| 選項 | 股票數量 | 所需時間 | 建議使用時機 |
|------|---------|---------|------------|
| 快速測試 | 20 檔 | 約 1 分鐘 | 初次使用確認功能正常 |
| 小型掃描 | 100 檔 | 約 5 分鐘 | **日常使用（推薦）** |
| 全市場掃描 | 約 950 檔 | 約 30-50 分鐘 | 每週一次深度掃描 |

> **注意：** FinMind 免費帳號（註冊會員）每小時限制 600 次 API 請求。系統背景會自動預先下載資料存入本機，掃描時優先讀本機快取，大幅減少 API 消耗。若額度不足，自動切換 yfinance 備援（法人條件暫停）。

### 篩選條件完整說明

#### ✅ 必要條件：全部達到才入選

**條件一：收盤價站上 20 日均線，且均線方向向上（佔 35 分）**

什麼是 20 日均線？
- 計算方式：最近 20 個交易日收盤價的算術平均值
- 白話意義：市場在過去一個月內的「平均成交成本」

為什麼要求均線「向上」？
- 僅僅站上均線不夠，均線本身必須是往上走的
- 如果均線是持平或向下，即使價格突破，也只是短暫的，容易拉回

本工具的具體判斷：
```
條件 A：今日收盤價 > 今日 MA20
條件 B：今日 MA20 > 5 個交易日前的 MA20
兩個條件都達到，才算通過
```

---

**條件二：今日成交量超過 5 日均量的 1.3 倍（佔 20 分）**

為什麼需要量能確認？

想像兩種情況：
- 情況 A：一檔股票突破均線，但今天交易量跟平常一樣
- 情況 B：一檔股票突破均線，今天交易量是平常的 2 倍

情況 B 代表這次突破有大量買家參與，說服力遠高於情況 A。沒有量能配合的突破，往往是假突破，很快就會跌回去。

1.3 倍的門檻設定邏輯：
- 太低（1.1 倍）：幾乎每天都達到，沒有篩選效果
- 太高（2 倍）：只有少數異常放量的股票才能入選，錯過太多機會
- 1.3 倍：比正常多三成，合理的「有效量」門檻

---

**條件三：MACD 黃金交叉或 RSI 在 50-70 健康區（佔 15 分）**

這個條件是「或」的關係，滿足其中一個即可。

**MACD 黃金交叉（優先確認）：**

```
黃金交叉定義：
  今日 DIF > DEA（快線穿越慢線往上）
  且昨日 DIF ≤ DEA（昨天快線還在慢線下方）

多頭延續（次要確認）：
  DIF > 0（在零線以上）
  且 DIF > DEA（快線仍高於慢線）
```

黃金交叉的交易含義：短期買盤力量（快線）超越了中長期均衡（慢線），市場動能從空轉多。

**RSI 50-70 健康區（備用確認）：**

為什麼選 50-70 而不選超賣（< 30）？

本工具的策略是「追強勢股」，而不是「撿便宜貨」：
- 超賣股（RSI < 30）雖然便宜，但可能有更深層的問題（業績下滑、主力出貨）
- RSI 在 50-70 代表股票正處於強勢狀態，但尚未過熱，還有繼續上漲的空間
- RSI > 70 代表超買，可能即將回調，此時進場風險偏高

---

**條件四：收盤價不低於布林通道下軌（佔 10 分）**

這是一個**排除弱勢股**的條件，而非買進理由。

布林通道下軌的意義：
- 下軌 = MA20 - 2個標準差
- 統計上，在正常情況下，只有約 2.5% 的交易日會跌破下軌
- 跌破下軌代表股票正在經歷「異常的大幅下跌」，遠超正常波動範圍

這類股票即使其他指標好看，也不入選，因為異常的賣壓背後通常有我們不知道的原因。

---

#### ➕ 加分條件：影響排名順序

**法人買超（佔 7 分）**

- 計算方式：外資 + 投信近 5 日累積淨買超（買超 - 賣超）> 0
- 意義：「聰明錢」在過去一週持續流入這檔股票
- 注意：自營商因為操作性質不同（避險需求多），不納入計算

**融資籌碼乾淨（佔 3 分）**

- 條件：最近的融資餘額低於之前的水準（呈現減少趨勢）
- 意義：持股者多為自有資金的穩定投資者，不容易因為短暫下跌而被迫賣出

---

#### ✨ 進階加分條件（可讓分數突破 100 分）

除了基礎條件，系統還會評估五項「進階強度指標」。這些加分可以讓分數超過 100，代表這是強勢股中的精選標的。

---

**進階條件一：週線多頭共振（額外 +10 分）**

什麼是週線多頭共振？

一般的技術分析只看「日線」——也就是每天的股價走勢。但專業交易員更重視「多週期共振」：當日線和週線（每週 K 線）同時都在多頭趨勢，進場的勝率會大幅提升。

本工具的具體判斷：
```
週線條件 A：本週收盤價 > 週線 MA10（週線均線方向的強度）
週線條件 B：週線 MA10 現在 > 4 週前的 MA10（週線趨勢向上）
兩個條件都達到，才給週線多頭加分
```

白話意義：這檔股票不只是「這幾天」在漲，而是從「更大的時間維度」就已經是上漲趨勢，代表機構法人和長線資金也在持續布局。

為什麼週線共振更可靠？

- 日線訊號雜訊多，容易出現假突破
- 週線代表的是「多週以上的趨勢共識」，被主力資金認可的時間更長
- 日線與週線同步向上，代表短、中期資金合力推升，訊號品質更高

可在側邊欄勾選「要求週線多頭」來強制只顯示週線也在多頭的股票（篩選更嚴格，但結果更精銳）。

---

**進階條件二：相對強度優於大盤（額外 +8 分）**

什麼是相對強度（Relative Strength, RS）？

RS 的概念源自美國投資大師 William O'Neil 的 IBD RS Rating 系統：
- 衡量這檔股票在過去 3 個月（63 個交易日）的漲幅，相對於大盤（加權指數）的表現
- RS 分數 0-100，分數越高代表這檔股票跑贏大盤越多
- > 70 分視為強勢股，代表這檔股票是市場中的資金磁鐵

本工具的具體判斷：
```
股票 3 個月報酬 = (現在收盤 - 63 天前收盤) ÷ 63 天前收盤 × 100%
大盤 3 個月報酬 = 加權指數同期報酬%
超額報酬 = 股票報酬 - 大盤報酬

RS 分數 = 50 + 超額報酬 × 2（上限 100，下限 0）

條件：RS 分數 ≥ 60 且跑贏大盤 → 給予加分
```

為什麼相對強度很重要？

想像大盤在過去 3 個月下跌了 5%，而一檔股票卻上漲了 8%：
- 這代表即使整體市場環境不好，仍有資金主動買進這檔股票
- 這種「抗跌性」通常代表法人或主力資金的支撐，是最強勢股票的特徵
- 當大盤回穩之後，這類股票往往是第一批「領漲」的

**「弱市抗跌、強市領漲」是 RS 高分股票的核心特質。**

RS 分數欄位：選股結果表格中會顯示每檔股票的 RS 分數。可在側邊欄設定「最低 RS 分數」來進一步過濾。

---

**進階條件三：均線多頭排列（額外 +5 分）**

什麼是多頭排列？

```
條件：MA5 > MA10 > MA20（三條均線由短到長依序向上排列）
```

三條均線完整排列代表短、中、長期的平均成本都在往上走，市場中各種持有期的投資人都在獲利，賣壓相對輕。這是趨勢最乾淨的狀態，訊號品質優於單純「站上 MA20」。

---

**進階條件四：量能優質（額外 +7 分）**

光有「量能放大」還不夠，更重要的是「量在哪裡放出來的」。

```
計算方式：近 10 個交易日中，上漲日的總成交量 ÷ 10 日總成交量
門檻：上漲日量能佔比 ≥ 60%
```

白話意義：這檔股票近期的大量，多數都出現在上漲的那幾天，代表買方積極、賣方保守。相反地，如果量大都出現在下跌日，反而是出貨訊號。

---

**進階條件五：突破盤整（額外 +8 分）**

什麼是突破盤整？

```
條件：今日收盤價 > 近 60 個交易日的最高收盤價（排除最近 3 日）
```

股票在盤整區間時，最高點就是「壓力位」——之前在那個位置有人賣出（獲利了結或解套），形成供給。當今天的收盤突破了這個壓力，代表那些賣壓已經被新進買盤消化吸收，上方阻力減少，是強勢突破訊號。

排除最近 3 日是為了避免「自我比較」——突破必須是突破「過去的壓力」，而不是跟昨天比。

---

### 過熱股防護機制（月線乖離率）

除了找強勢股，系統也新增了「不要追得太高」的保護機制，避免買到短線漲幅過大、隨時可能急跌拉回的過熱股。

**什麼是月線乖離率？**

```
月線乖離率 = (今日收盤價 - 今日 MA20) ÷ 今日 MA20 × 100%
```

白話意思：目前股價比 20 日均線高出多少百分比。

例如：
- MA20 = 100 元
- 今日收盤 = 115 元
- 月線乖離率 = +15%

這代表股價已經明顯脫離近一個月的平均成本區，雖然仍然強勢，但短線拉回風險也同步上升。

**Sidebar 可設定兩個參數：**

1. **最大容許乖離率 (%)**
- 預設值：15%
- 可調整範圍：5% 到 30%

2. **超過門檻時的處置方式**
- **直接剔除（不入選候選清單）**
  適合保守型使用者，不想追價過熱股
- **扣分懲罰（總強度分數扣減 10 分）**
  適合仍想保留觀察名單，但讓過熱股排名自然往後

**建議如何使用？**

- 多頭初升段、偏保守操作：用 **直接剔除**
- 主升段行情、仍想保留最強標的：用 **扣分懲罰**
- 若最近市場波動很大、常見急拉急殺：可把上限從 15% 下修到 10-12%

> **提醒：** 高乖離率不代表一定不能買，而是代表「追高風險提高」。這個機制的目的不是否定強勢股，而是避免在最容易拉回的位置追價。

---

### 分數解讀與操作建議

| 強度分數 | 解讀 | 建議 |
|---------|------|------|
| **> 100 分** | **強勢精選**：基礎條件全過，且有多項進階加分確認 | **最優先研究，是當前市場中最強的一批股票** |
| 80-100 分 | 所有基礎條件都符合，有部分進階加分 | 優先研究，看 K 線圖確認後考慮進場 |
| 65-79 分 | 技術條件完整，進階加分較少 | 值得關注，確認大盤環境後評估 |
| 50-64 分 | 剛好達到所有必要條件 | 列入觀察清單，暫不急於進場 |

> **特別說明：** 分數滿分上限為 138 分（所有進階條件全部觸發）。超過 100 分不是系統錯誤，代表這批股票在技術面、籌碼面、多週期、相對強度、形態等各個維度都同時得到確認，是當前市場中最稀有的強勢股。

| 欄位 | 意思 | 如何解讀 |
|------|------|---------|
| 強度分數 | 0-138，越高越強 | 優先看 65 分以上；> 100 為強勢精選 |

| 觸發條件 | 哪些條件被達到 | 一目了然地看到入選理由 |

### 產業族群分析（第二個頁籤）

除了個股排行榜，選股雷達還提供「產業族群分析」功能，幫你看懂資金流向。

**什麼是產業族群分析？**

當選股雷達同時選出 3 檔散熱模組股票、2 檔 AI 伺服器股票，這代表資金正在集中流入這兩個產業。這種現象叫做「族群輪動」——機構法人通常以「產業佈局」的方式進場，而不是隨機選股。

**如何解讀？**

| 顯示資訊 | 意義 |
|---------|------|
| 各產業入選檔數 | 入選越多，代表這個產業正在吸引更多資金 |
| 產業平均分數 | 分數越高，代表這個產業的整體技術面越強 |
| 各產業最強股 | 在最熱門的產業中，分數最高的就是最值得研究的標的 |

**操作建議：**

1. 找出入選最多的 1-2 個產業
2. 在那個產業中，選分數最高的 1-2 檔股票深入研究
3. 「最強產業裡的最強股票」是機構法人的基本操作邏輯

---

### 流動性前置過濾（掃描前先縮小範圍）

在開始掃描前，可以先用流動性條件過濾股票宇宙，聚焦真正值得關注的標的：

| 模式 | 說明 | 建議使用時機 |
|------|------|------------|
| **前日量前 N 名**（預設 100）| 鎖定昨天成交最活躍的 100 檔股票 | 日常使用，動態追蹤市場熱點 |
| **日均量 ≥ N 張**（預設 1000，上限 50,000）| 近 20 日平均量超過 N 張才考慮 | 想篩掉低流動性股票時使用 |
| 不過濾 | 掃描全部股票 | 週末深度掃描 |

> **為什麼用量能代替資本額？** 日均量超過 1000 張的股票，資本額幾乎必然超過 20 億，兩個條件高度重疊。而量能是每日動態更新的，比靜態的資本額更能反映當下市場關注度。

---

### 產業輪動過濾（只掃最強族群）

在流動性過濾之上，還可以再加一層「產業輪動過濾」：

**原理：** 計算所有已下載股票的近 5 日（一週）漲幅，按產業分組取平均，選出漲幅排名前 N 個產業，只在這些產業內做技術篩選。

**操作：** Sidebar 勾選「只掃描近一週漲幅前 N 類股」，預設取前 3 個產業。

**掃描完成後頁面會顯示：**
```
📊 本次鎖定產業（近 5 日漲幅前 3 名）：
  半導體 (+4.2%)　散熱模組 (+3.8%)　AI伺服器 (+3.1%)
```

**三層過濾疊加效果：**
```
全市場 950 檔
  → 前日量前 100 名（流動性過濾）
  → 近一週漲幅前 3 個產業（產業輪動過濾）
  → 技術分析篩選（必要條件全過）
  → 最終候選：5-15 檔精選個股
```

這種由上而下（Top-Down）的篩選邏輯，和機構法人的選股流程一致：**先選大環境 → 再選產業 → 最後選個股**。

### 掃描歷史紀錄（第四個頁籤）

每次掃描完成後，系統會自動將結果儲存到本機資料庫。在「歷史紀錄」頁籤中可以：

| 功能 | 說明 |
|------|------|
| 查看過去記錄 | 最新的在最上方，顯示掃描時間與找到的股票數量 |
| 展開查看條件 | 可看到當時使用的所有篩選條件（股價、量能、產業、RS 門檻等） |
| 結果預覽 | 展開後直接顯示前 10 筆股票 |
| 載入結果 | 把歷史結果還原到「掃描結果」頁籤，方便重新查看或比較 |
| 刪除紀錄 | 可刪除不再需要的歷史 |

**用途：** 可以回頭比較不同日期的掃描結果，觀察股票從候選到真正啟動的時間差，慢慢建立自己對「訊號出現到進場時機」的感覺。

---

### 篩選漏斗（🔬 第二個頁籤）

掃描完成後，「篩選漏斗」頁籤會顯示每個條件在篩選過程中排除了多少股票，讓你清楚看到條件的嚴格程度。

**主要顯示內容：**

| 區塊 | 說明 |
|------|------|
| 漏斗長條圖 | 從「載入資料」到「通過所有條件」的逐層存活數量 |
| 各條件個別命中率 | 每個技術條件單獨觸發的比例（不是累積排除） |
| 近失標的（Near Miss）| 差一兩個條件就入選的股票，依缺失條件分組展示 |
| 前置過濾排除明細 | 因股價過低、日均量不足、資料不足而被排除的股票統計 |

**如何利用漏斗優化條件：**
- 若漏斗中某個條件排除了 90% 以上的股票 → 可能設得太嚴，考慮放寬
- 若某個加分條件命中率只有 5% → 對排名影響有限，可評估是否保留
- 「近失標的」是很好的觀察清單——這些股票幾乎符合，可手動複查是否值得追蹤

### 基本面過濾（財報品質篩選）

在 Sidebar 的「基本面過濾」區塊，可以選擇啟用財報條件，在技術篩選之前先排除虧損股與財務地雷。

| 條件 | 說明 |
|------|------|
| EPS TTM > 0 | 近四季合計每股盈餘為正（排除虧損公司） |
| 營業現金流 > 0 | 近四季合計營業現金流為正（排除假獲利） |
| 最低 ROE | 近四季股東權益報酬率下限（0 = 不限，建議 10-15%） |
| 負債比上限 | 最新季負債比例上限（0 = 不限，一般產業建議 < 60%） |

**快取機制：** 財報資料快取 90 天，首次啟用時需呼叫 API，之後讀本機快取，幾乎零額度消耗。

> **注意（免費帳號限制）：** FinMind 財報端點（`TaiwanStockFinancialStatements`）需要**付費方案**。免費帳號啟用基本面過濾時，系統會顯示橘色提示，自動跳過基本面篩選，選股結果不受影響。

### 族群集體突破偵測（🔥 新功能）

在 Sidebar「族群集體突破偵測」區塊可啟用兩個進階的族群動量指標，掃描後在「產業族群」頁籤的最底部顯示結果。

#### 指標一：族群創高密度（HP Density）

**原理：** 計算族群內有多少比例的股票今日收盤等於近 N 日最高收盤。

```
族群 A 有 20 檔股票
其中 8 檔今天收盤創了 20 日新高
→ HP Density = 8 ÷ 20 = 40%
```

**意義：** 一個族群裡有 40% 以上的股票同時創高，代表這個族群是「集體突破」，而非個別股票的偶然創高。這種族群性的動作通常代表機構法人正在整族群佈局。

**可調整參數：**
- 創高天數 N（預設 20 日）
- 族群創高比例門檻（預設 30%）

#### 指標二：資金流向比重（Turnover Ratio）

**原理：** 以「收盤價 × 成交量」估算各族群的成交額，計算其佔全市場的比重，找出資金最集中的族群。

```
族群 B 今日估算成交額 = 80 億
全市場估算成交額 = 2000 億
→ 資金比重 = 80 ÷ 2000 = 4%（排名第 2）
```

**意義：** 錢往哪裡流，股票就往哪裡漲。資金比重排名前 N 的族群，代表今天市場最積極追逐的方向。

**可調整參數：**
- 資金前幾大族群（預設 5）

#### 集體突破判斷邏輯

| 啟用指標 | 判斷方式 |
|---------|---------|
| 只啟用 HP Density | 創高比例 ≥ 門檻 → 🔥 集體突破 |
| 只啟用 Turnover | 成交額排名 ≤ N → 🔥 集體突破 |
| 兩個都啟用 | 兩個條件同時達到 → 🔥 集體突破 |

### 籌碼面條件：法人買超（可自訂模式）

法人條件現在支援兩種模式，在 Sidebar 的「納入法人買賣超」區塊操作：

**模式 A：個別法人皆須買超（嚴格模式）**

可自由勾選外資、投信、自營商，系統會檢查：

- 勾選的法人中，**每一個**都必須在**連續 N 個交易日**為淨買超
- 任何一方賣超，或連續天數不足，都不算達標

這個模式適合你想找「多方資金共識高」的股票。

**模式 B：三大法人合計買超（寬鬆模式）**

這個模式不要求三方各自都買超，而是看整體資金是否淨流入。可選兩種判斷方式：

- **近 N 日累計 > 0**：最近 N 個交易日的三大法人合計淨買超總和大於 0
- **連續 N 日每日 > 0**：最近 N 個交易日中，每一天的三大法人合計淨買超都大於 0

這種模式可減少自營商避險單對訊號的干擾，更適合觀察整體資金方向。

**法人加分規則**

| 模式 | 觸發條件 | 加分 |
|------|---------|------|
| 個別法人皆須買超 | 勾選法人各自連續買超 | +7 |
| 三大法人合計買超 | 合計淨買超達標 | +4 |
| 三大法人合計買超 + 外資投信同步買超 | 合計達標，且外資與投信同時連續買超 | +7（= 4 + 3） |

**必要條件 vs. 加分條件**

- **加分條件（預設）**：符合法人條件才加分，不符合不影響入選
- **必要條件**：勾選「法人條件為必要條件」後，不符合的股票直接剔除

建議：
- 想找「有明確法人背書」的強勢股 → 用「個別法人皆須買超」並設為必要條件
- 想找「整體資金在流入」的股票 → 用「三大法人合計買超」

### 正確使用選股雷達的步驟

```
第一步：先看「市場環境」頁面，確認大盤現在是多頭還是空頭
第二步：空頭環境 → 選股雷達結果僅供參考，不建議進場
         多頭環境 → 進行以下步驟
第三步：在 Sidebar 設定掃描條件：
         - 流動性過濾：建議選「前日量前100名」
         - 產業輪動：建議勾選「前3個產業」
         - 有法人資料需求時，依風格選擇「個別法人皆須買超」或「三大法人合計買超」
         - 若不想追價過熱股，可開啟「過熱股防護機制」，預設上限 15%
第四步：執行掃描，找出分數 65 分以上的股票
第五步：看「產業族群」頁籤，確認哪個產業最強
第六步：在最強產業中，點入分數最高的個股查看 K 線圖，確認：
         - 趨勢方向清晰嗎？日線、週線都在多頭嗎？
         - 突破盤整了嗎？量能是否集中在上漲日？
         - RS 分數是否 > 70（是否在市場中屬於強勢股）？
第七步：找到合理的進場點和停損位置
第八步：前往「風險控制」計算應該買幾張
第九步：進場後立刻在「持股監控」新增這筆持股
```
