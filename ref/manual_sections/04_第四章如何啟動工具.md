## 第四章：如何啟動工具

### 建議啟動方式：Srock CLI

新版提供 `srock` 命令列管理工具，用來統一啟動本機 Dashboard、Basic Auth 反向代理與 Tailscale Funnel。第一次使用若系統還找不到 `srock` 指令，請先在專案資料夾執行一次：

```
cd "E:\workspace\srock tool"
python -m pip install -e .
```

之後可用以下指令操作：

| 指令 | 用途 |
|------|------|
| `srock status` | 查看 Streamlit、Auth Proxy、Funnel 目前狀態 |
| `srock up --profile local` | 只啟動本機 Streamlit（開發或只在自己電腦使用） |
| `srock up --profile protected` | 啟動 Streamlit + Caddy Basic Auth，但不開外網 |
| `srock up --profile full` | 啟動 Streamlit + Caddy Basic Auth + Tailscale Funnel，對外提供受保護入口 |
| `srock down` | 依序關閉 Funnel、Auth Proxy、Streamlit |
| `srock restart --profile protected` | 重啟受保護的本機入口 |
| `srock watch` | 即時監控服務狀態 |
| `srock logs all` | 查看 Streamlit / Caddy 最近 log |
| `srock open` | 開啟本機 Streamlit |
| `srock open --public` | 開啟目前 Funnel 對外網址 |

如果你只想快速啟動預設設定，也可以雙擊專案根目錄的 `srock-up.bat`。它會進入專案資料夾並執行 `srock up`。

### 服務啟動完成通知

使用 `srock up` 或 `srock restart` 啟動服務時，系統會等 Streamlit、Auth Proxy 與 Cloudflare Tunnel 依照 profile 啟動完成後，推播一則 LINE 訊息給所有啟用中的訂閱者。若你分開手動啟動元件，最後執行 `srock start funnel` 並取得 Cloudflare 對外網址後，也會送出同一種啟動通知。訊息內容包含：

- 服務已啟動
- 啟動完成時間
- 目前可用網址；`full` profile 會優先顯示 Cloudflare Tunnel 對外網址，`protected` / `local` 則顯示本機網址

為避免半夜或清晨打擾，啟動通知只會在本機時間 **08:00 到 20:00** 之間送出。時間外啟動服務仍會正常啟動，只是不推播啟動訊息。這個通知只掛在 CLI 啟動流程，不會因為 Streamlit 頁面重新整理而重複推播。

### Basic Auth 帳號管理

若要開啟受保護入口，請先建立至少一組帳密：

```
cd "E:\workspace\srock tool"
srock auth add srock
```

系統會要求你輸入密碼並確認。帳號與 bcrypt 雜湊會存在 `secrets/basic_auth_users.json`，Caddy 設定會寫入 `secrets/Caddyfile`；這些檔案已被 `.gitignore` 排除，不會提交到版本庫。

| 指令 | 用途 |
|------|------|
| `srock auth list` | 列出目前 Basic Auth 帳號 |
| `srock auth add <username>` | 新增帳號 |
| `srock auth reset <username>` | 重設帳號密碼；帳號不存在時會建立 |
| `srock auth delete <username>` | 刪除帳號 |

> **注意：** Caddy 執行檔預設位置為 `tools/caddy/caddy.exe`，Tailscale Funnel 需要已安裝 Tailscale 並登入。若只要本機開發，使用 `srock up --profile local` 即可，不需要 Caddy 或 Tailscale。

### 傳統啟動 Dashboard（仍可使用）

1. 打開黑色視窗（搜尋「命令提示字元」或「PowerShell」）
2. 輸入以下指令：

```
cd "E:\workspace\srock tool"
streamlit run app.py
```

3. 瀏覽器會自動打開，看到主頁畫面就代表成功
4. 左側有選單，可切換各功能頁面

### 啟動 LINE 自動推播（盤後任務）

另開一個黑色視窗，輸入：

```
cd "E:\workspace\srock tool"
python scheduler/jobs.py
```

> **注意：** 這個視窗要一直開著，電腦不能關機。如果關機或關閉視窗，當天的自動推播就不會發送。
> 盤中持股分K監控已改為 App 內建排程器，不需要另外開這個外部排程視窗；請到「持股監控」頁啟動。
