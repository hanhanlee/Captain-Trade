## 第十三章：LINE 自動推播設定

### 推播種類與時間

| 時間 | 推播內容 | 觸發條件 |
|------|---------|---------|
| 週一至週五 13:30 | 持股盤中警示 | 有任何警示訊號才發送 |
| 週一至週五 14:35 | 持股盤後警示 | 有任何警示訊號才發送 |
| 週一至週五 14:45 | 選股雷達結果前 5 名 | 每日固定發送 |
| 週五 15:10 | 週績效摘要 | 每週固定發送 |

### 推播格式範例

**選股雷達推播：**

```
📊 今日選股雷達 — 強勢候選股

1. 2330 台積電
   收盤：850 元  🔴+2.3%
   量比：1.8x  分數：85
   [站上MA20、MA20向上、量增、MACD黃金交叉、布林正常]

2. 2317 鴻海
   ...
```

**持股警示推播：**

```
💼 持股監控警示

🔴 2330 台積電
   跌破停損價 780 元
   現價 775 元  損益 -8.2%

🟡 2454 聯發科
   從高點回撤 9.5%
   現價 820 元  損益 +3.1%
```

### 手動觸發推播

如果不想等自動時間，可以在「市場環境」頁面的「排程控制」頁籤，隨時手動觸發任何推播任務。
