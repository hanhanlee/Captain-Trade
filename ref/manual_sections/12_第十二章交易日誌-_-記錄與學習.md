## 第十二章：交易日誌 — 記錄與學習

### 為什麼記錄比選股更重要

傳奇操盤手 Paul Tudor Jones 說：「不分析自己的交易，等於在黑暗中射箭。」

不記錄的投資人，會不斷重複同樣的錯誤而渾然不知。記錄的目的不是懲罰自己，而是找到規律：
- 哪類交易你最擅長？
- 什麼情緒下你最容易做錯誤決定？
- 你的真實勝率和盈虧比是多少？

### 每筆交易需要記錄的內容

| 欄位 | 必填 | 說明 |
|------|------|------|
| 股票代碼 | 是 | 四碼代號 |
| 動作 | 是 | BUY（買進）或 SELL（賣出） |
| 價格 | 是 | 實際成交價 |
| 張數 | 是 | 幾張 |
| 交易日期 | 是 | 實際成交日期 |
| 進出場理由 | **強烈建議** | 你為什麼買進 / 賣出（最重要的欄位） |
| 情緒標記 | 建議 | 這次決策時的心理狀態 |
| 損益 | 賣出時填 | 這筆交易實際賺或虧了多少元 |

### 情緒標記的選項與意義

| 情緒 | 描述 | 常見的錯誤 |
|------|------|-----------|
| 冷靜 | 按照計畫執行 | — |
| 樂觀 | 對後市有信心 | 可能低估風險 |
| 貪婪 | 想賺更多而加碼 | 放棄停損、擴大部位 |
| 恐慌 | 害怕繼續虧損 | 在低點砍在最低 |
| 衝動 | 沒有分析就進場 | 沒有計畫的買進 |
| 猶豫 | 一直在觀望 | 好機會出現卻沒進場 |
| FOMO | 怕錯過而追高 | 在過高的位置買進 |

長期記錄後，你會看到哪種情緒下的勝率最低，這就是你的心理弱點，需要特別注意。

### 三個核心績效指標

**指標一：勝率**

```
勝率 = 獲利交易次數 ÷ 總交易次數 × 100%
```

常見誤解：很多人以為要勝率 > 50% 才算好。實際上，只要盈虧比夠高，40% 的勝率也能穩定獲利。

**指標二：盈虧比（Profit Factor）**

```
盈虧比 = 所有獲利交易的總獲利 ÷ 所有虧損交易的總虧損
```

| 盈虧比 | 評估 |
|-------|------|
| < 1 | 虧損的總額大於獲利，長期必虧 |
| 1 - 1.5 | 僅達損益平衡，還需改善 |
| **1.5 - 2** | **良好，可以持續獲利** |
| > 2 | 優秀 |

**指標三：期望值**

```
期望值 = 勝率 × 平均獲利 - (1 - 勝率) × 平均虧損
```

期望值 > 0 = 這個策略長期下來會獲利。

以下用一個表格展示不同勝率和盈虧比組合的期望值：

| 勝率 | 盈虧比 | 每次期望值（假設虧損 10,000 元） |
|------|-------|------------------------------|
| 30% | 3:1 | +1,200 元 ✅ |
| 40% | 2:1 | +4,000 元 ✅ |
| 50% | 1.5:1 | +2,500 元 ✅ |
| 55% | 1:1 | +1,000 元 ✅ |
| 60% | 0.8:1 | -800 元 ❌ |

**重點：高勝率不代表獲利，盈虧比才是決定性因素。**
